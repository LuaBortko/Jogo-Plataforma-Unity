# Plataformer-Unity
Projeto desenvolvido na Unity como atividade de laboratório, consistindo na criação de um jogo simples de plataforma.

## Requisitos do Projeto
- Jogo do tipo plataforma.
- Tela inicial com orientações diretas de como jogar.
- Duas fases. (cenas)
- Tratamento de colisão com as plataformas.
- Fundo com parallax scrolling conforme o jogador se move.
- Jogador, inimigos e itens com sprites animados.
- Itens colecionáveis. Ex: moedas, vidas, maçãs, bananas etc.
- Pontuação
- Cena final do jogo informando derrota ou vitória com botão para reiniciar.
  - O jogador perde se os inimigos machucarem o jogador e as vidas chegarem a zero.
  - O jogador ganha se coletar todos os itens predeterminados.
